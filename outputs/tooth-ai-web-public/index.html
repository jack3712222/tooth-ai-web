訓練與優化器

YOLO 訓練可設定的超參數很多；本專案固定使用 AdamW，並將其中 5 個欄位納入 grid search：box、lr0、weight_decay、momentum、scheduler。實驗空間為 2 × 3 × 3 × 2 × 2 = 72 組，每組 10 runs，共 720 次結果彙整。

官方訓練設定依據

Ultralytics Train 文件將 batch size、learning rate、momentum、weight decay、optimizer、loss function 與資料組成列為會影響訓練表現的重要設定。本頁只呈現本專案實際使用到的設定。

AdamW optimizer 參數

AdamW 是 PyTorch optimizer；Ultralytics 會把 YOLO 訓練參數映射到 optimizer。此處分成 PyTorch AdamW 原生參數，以及本專案實際在 YOLO 設定中可見/有調整的欄位。

AdamW 參數PyTorch 預設YOLO 對應/本專案說明
params模型參數YOLOv8m trainable weights要被最佳化的模型權重。
lr0.001lr0：0.0001 / 0.001 / 0.01學習率，本次有搜尋。
betas(0.9, 0.999)momentum 對應 beta1：0.900 / 0.937一階/二階動量係數；YOLO 文件說 momentum 對 Adam 類為 beta1。
eps1e-8未另外設定數值穩定項。
weight_decay0.010.001 / 0.010 / 0.100AdamW 的 decoupled weight decay，本次有搜尋。
amsgradFalse未另外設定是否使用 AMSGrad 變體。
maximizeFalse未另外設定一般訓練是最小化 loss。
foreach / fusedNone由 PyTorch/環境決定影響 optimizer 實作效能，不是本次搜尋欄位。
capturable / differentiableFalse / False未另外設定圖捕捉或可微分 optimizer 設定。
